Como é feito o Onboarding de APIs na GIC?

WF API PIX - Clientes - Certificado Integrador (1 - n)

Viabilizar a integração da API Pix Banrisul por clientes do segmento varejo que utilizem soluções de integradores que autenticam a conexão com a API usando seu próprio certificado.
Workflow API Pix - Clientes

Viabilizar a integração da API Pix Banrisul por clientes do segmento varejo.
WF Chave PIX e Credenciais

Criação de chave Pix e credenciais para acesso ao Portal do Desenvolvedor Banrisul para clientes da Unidade Comercial de Governos.

cliente/aplicação - APIM
